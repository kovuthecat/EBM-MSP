## 3. Risque de biais (cocher selon le design)

### Si ECR (inspiré RoB2)
- [ ] Randomisation adéquate (séquence + allocation dissimulée)
- [ ] Aveugle (patients / soignants / évaluateurs) — sinon, critère objectif ?
- [ ] Données de sortie complètes (perdus de vue faibles, analyse **en ITT**)
- [ ] Pas de **sélection du résultat rapporté** (critère principal = celui pré-enregistré)
- [ ] Pas d'**arrêt précoce** pour bénéfice (sinon, surestimation probable)

### Si méta-analyse / revue systématique (inspiré AMSTAR-2)
- [ ] Question et critères d'inclusion pré-établis (protocole/PROSPERO)
- [ ] Recherche exhaustive (≥ 2 bases, littérature grise)
- [ ] Évaluation du risque de biais des études incluses
- [ ] Hétérogénéité analysée et discutée (I²)
- [ ] Biais de publication évalué (funnel plot, etc.)

### Si observationnel (cohorte / cas-témoins)
- [ ] Facteurs de confusion identifiés et **ajustés**
- [ ] Causalité inverse envisagée
- [ ] Groupes comparables à l'inclusion
- [ ] Signaler : **niveau de preuve d'emblée plus faible** (association ≠ causalité)

**Synthèse risque de biais :** faible / modéré / élevé — justification :

## 4. Critère de jugement (rubrique la plus discriminante)

- Critère principal : ______
- **Dur** (mortalité, IDM, AVC, IRT, cécité, fracture…) **ou substitution** (HbA1c, LDL, albuminurie, TA, marqueur d'imagerie…) ? → **dur / substitution**
- Composite ? Si oui, **quel composant porte le résultat** ? ______
- Le critère est-il **pertinent pour le patient** ? oui / non

> Vigilance : un bénéfice sur un critère de substitution **n'implique pas** un bénéfice clinique.

## Exemple minimal (repère)

> **Étude X, ECR, cible LDL intensive vs modérée, prévention secondaire.** Critère : composite CV (**dur**), porté par les revascularisations. Effet : HR 0,67 ; **NNT 33 à 3 ans**. Risque de biais faible, mais confusion cible/molécule (bras intensif = plus de rosuvastatine 20 + ézétimibe). Validité externe correcte. Cohérent avec le CTT mais réduction de risque supérieure à ce que la seule baisse de LDL prédit → **effet possiblement lié aux molécules**. **Niveau : modéré.** Impact : **pratique** (cardio-vasculaire, MG/IPA). Impacte le nœud « statine/cible » → à soumettre au comité.
